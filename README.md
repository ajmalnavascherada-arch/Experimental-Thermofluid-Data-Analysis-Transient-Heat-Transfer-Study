# Experimental Thermofluid Data Analysis

## Overview
This project analyzes transient thermal behavior in a system using experimental-like data. The focus is on understanding heat transfer characteristics through temperature evolution and its relationship with operating conditions.

## Objectives
- Analyze temperature variation over time
- Compute temperature gradients (dT/dt)
- Investigate correlation between current and thermal response
- Automate data processing and visualization

## Methods
- Data preprocessing and unit conversion
- Numerical differentiation
- Correlation analysis
- Visualization using matplotlib

## Key Results
- Temperature increases with current, indicating heat generation effects
- Positive correlation between current and temperature
- Gradient analysis highlights transient thermal behavior

## Tools Used
- Python (pandas, numpy, matplotlib)

## Structure
- `data/`: input dataset  
- `src/`: analysis modules  
- `results/`: outputs and plots
